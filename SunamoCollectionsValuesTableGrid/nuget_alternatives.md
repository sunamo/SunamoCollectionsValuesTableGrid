# NuGet Alternatives to SunamoCollectionsValuesTableGrid

This document lists popular NuGet packages that provide similar functionality to SunamoCollectionsValuesTableGrid.

## Overview

Table/grid data structure utilities

## Alternative Packages

### System.Data.DataTable
- **NuGet**: System.Data.Common
- **Purpose**: Built-in table structure
- **Key Features**: Rows, columns, constraints, relations

### CsvHelper
- **NuGet**: CsvHelper
- **Purpose**: CSV reading/writing
- **Key Features**: Type conversion, configuration, streaming

### EPPlus
- **NuGet**: EPPlus
- **Purpose**: Excel file manipulation
- **Key Features**: Read/write Excel, formulas, styling

### ClosedXML
- **NuGet**: ClosedXML
- **Purpose**: Excel library
- **Key Features**: Fluent API, tables, ranges, formatting

## Comparison Notes

DataTable for in-memory tables. EPPlus/ClosedXML for Excel. CsvHelper for CSV.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **System.Data.DataTable**: Built-in table structure
- **CsvHelper**: CSV reading/writing
- **EPPlus**: Excel file manipulation
